安装虚拟机

[官网下载地址];(https://www.vmware.com/info/workstation-player/evaluation )

第一步打开创建虚拟机

第二部

默认经典点下一步

第三部

选择安装程序光盘映像文件， 没有镜像的可以旁边系统镜像下载

第五部

填产品密钥或直接跳过填不填都行，我这边就直接下一步

第六部

选择系统安装位置我这一边就改位置C盘大的就默认点下一步



)

第七部

调整虚拟机磁盘大小默认是60gb不够用的可以往上调其他默认就行调完点击下一步

第八部

点完成等待磁盘配置好进入系统安装界面

第九部

来到系统选择界面为了更好的可操作性我选专业版其他人可以根据个人情况选择适合自己的版本

第十部

安装vmware Tools都是自动的安装的一路同意就行了
遇到一下情况就点第二个始终安装此驱动程序软件就行
